#          Openjdk for Termux 
## install
1.更新源
```bash
pkg update && pkg upgrade
```
2.安装wget
```bash
pkg install wget -y
```
3.下载一键安装脚本
  3.1 安装openjdk11
  ```bash
  wget https://github.com/zryyoung/openjdk-Termux/blob/main/install_jdk11.sh
  ```
  3.1.1 执行脚本
  bash install_jdk11.sh
